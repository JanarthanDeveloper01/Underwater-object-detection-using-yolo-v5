# 🌊 Underwater Object Detection using YOLOv5

## 📌 Project Overview

This project focuses on **real-time underwater object detection** using the YOLOv5 deep learning model.
It is designed to identify marine species such as **starfish, echinus, scallops, and other underwater objects**.

---

## 🚀 Features

* Real-time object detection
* Trained using YOLOv5 architecture
* Custom underwater dataset (Roboflow)
* High accuracy detection of marine species
* Scalable for real-world marine applications

---

## 🧠 Technologies Used

* Python
* YOLOv5 (PyTorch)
* OpenCV
* Roboflow API
* Jupyter Notebook

---
